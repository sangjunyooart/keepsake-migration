# Handoffs

Create one markdown file per task or work session that another contributor may need to continue.

Suggested filename:

`YYYY-MM-DD-short-task-name.md`

Template:

```md
# Task

## Summary
- What changed

## Files touched
- path/to/file

## Verification
- Commands or manual checks performed

## Risks / open questions
- What the next contributor should watch

## Next step
- First recommended follow-up action
```

Keep the note short and factual. Link the PR when one exists.
